The University of Lorraine, like most European Universities and some Universities around the world, uses the Eduroam network for internet access. It is also used in most of the residences here.

Eduroam needs a username and a password. If you already have an Eduroam account from a previous university, you can keep using them.

Setting up Eduroam in a device can be either really simple or really difficult.

What account to use?

After registering at the University of Lorraine you should have activated your studend account. Afther that, you should receive two email aliases for your account: A short one and a long one.

The login to eduroam must be done with the short alias!

Note: When you just arrive to your residence you can request a temporary account for Eduroam at the reception.

Windows, Mac, Android, iOS

For these platform, setting up eduroam is pretty straight forward: Just connect to the "eduroam" network in your WiFi settings, and you'll be asked for your username and password. Then you are done!

Linux

Connecting to eduroam with Linux is painful.

Option 1: Eduroam CAT

Go to the website of the Eduroam CAT (Eduroam Configuration Assistant Tool).

You will need to download a configuration script from this website.

  • On the main page there is a button for downloading a configuration script
  • You will be asked for your institution, choose "UniversitÃ© de Lorraine" (or "CROUS Lorraine" if you are using an temporary account from your residence)
  • Then, you will be able to download the script, which is a Python (3) file.

Apparently, the configuration script changes every year, so there is no point in me putting it here.

To configure eduroam, run this script on a terminal. You will eventually be asked your username and password. For your username, use your full email, i.e., user@server.

Cross your fingers and hope it works! You know how Linux is....

Option 2: Manual configuration

The above script does not work on some Linux distributions (like Mint).

Manual configuration is the last resort.

You should follow the instruction in this documentation website from the University of Lorraine, where you will have to download a CA (Certification Authorities) certificate file and manually input the parameters for connecting to the network.
